# Security Policy

## Reporting a Security Issue

This repository contains documentation for a cybersecurity capstone project. All data and findings are fictional and used for educational purposes.

If you discover a security issue with the documentation or repository itself, please report it by opening a GitHub issue tagged `security`.

## Sensitive Data Handling

- Do **not** commit real credentials, real PII, or production system configurations to this repository
- All company names, system names, and personal data in this repository are fictional
- Evidence files containing real data should be stored in access-controlled systems, not public repositories

## Repository Classification

This repository is classified as: **Public — Educational Content Only**

All findings, risk scores, and compliance assessments are based on a simulated fictional organization.
