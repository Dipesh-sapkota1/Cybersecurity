# Cybersecurity Incident Timeline Analysis

## Project Overview

This project analyzes five major cybersecurity incidents that significantly influenced modern cybersecurity practices and defensive strategies.

The report explores:
- Attack types
- Real-world impact
- Failed security controls
- Security improvements introduced afterward
- Lessons learned from historical cyber incidents

---

## Attacks Covered

1. Morris Worm (1988)
2. ILOVEYOU Worm (2000)
3. Target Breach (2013)
4. WannaCry Ransomware (2017)
5. SolarWinds Supply Chain Attack (2020)
